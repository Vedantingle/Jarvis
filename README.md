# Jarvis - AI Voice Assistant 🎙️  

Jarvis is a Python-based AI voice assistant that automates daily tasks through voice commands. Inspired by Marvel’s Jarvis, this project brings hands-free control to your desktop, allowing you to manage applications, fetch information, and perform system operations seamlessly.  

## 🔧 Features  
- **Voice Recognition:** Execute tasks by speaking commands.  
- **Time & Date:** Get real-time updates.  
- **Music Player:** Play music from your local library.  
- **Web Automation:** Open Google, YouTube, and more.  
- **Screenshot Tool:** Capture your screen with a command.  
- **Text Writing:** Dictate text in real-time.  
- **Volume Control:** Adjust, mute, or increase system volume.  
- **Wikipedia Search:** Fetch quick information from Wikipedia.  
- **Fun Interaction:** Enjoy jokes and light-hearted interactions.  
- **System Management:** Shutdown, restart, or exit with voice prompts.  

## 🛠️ Tech Stack  
- **Python** – Core language.  
- **pyttsx3** – Text-to-speech conversion.  
- **SpeechRecognition** – Recognize voice commands.  
- **pyautogui** – Automate keyboard/mouse actions.  
- **Wikipedia API** – Fetch summaries from Wikipedia.  

## 🚀 Installation  
1. Clone this repository:  
   ```bash
   git clone https://github.com/yourusername/Jarvis-Voice-Assistant.git
   ```  
2. Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```  
3. Run the assistant:  
   ```bash
   python voice_assistant_web/app.py
   ```  

## 🗂️ File Structure  
- `app.py` – Main application entry point.  
- `assistant.py` – Core logic and command processing.  
- `requirements.txt` – List of dependencies.  

## 🤖 Usage  
- Speak clearly after the prompt “Listening...”  
- Example Commands:  
  - **"What's the time?"**  
  - **"Take a screenshot"**  
  - **"Play music"**  
  - **"Tell me a joke"**
